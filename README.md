# Client Lead Management System

This project is a simple Client Lead Management System developed using HTML, CSS, and JavaScript as part of my **Future Interns Web Development Internship (Task 2)**.

The purpose of this project is to help users manage client leads in a simple and organized way. Users can log in, add new leads, search for existing leads, edit lead details, delete leads, and manage all lead information through a clean dashboard.
